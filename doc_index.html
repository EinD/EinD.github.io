Documentation

The following resources are available online for users and developers of EinD:

Users

Installation Instructions

EinD is currently under development, and no releases have been published yet. To install a pre-alpha, experimental version of EinD, please refer to the installation instructions in the EinD Wiki.

User Manual

If you have installed EinD, execute the :help command to access its user manual. For an overview of features specific to EinD, see the introduction to Nvim (:help nvim-intro). This also contains instructions on how to enable support for Python plugins (:help nvim-python) such as YouCompleteMe.

Other helpful resources for both EinD and Vim are the quick reference guide (:help quickref) and the Vim FAQ.

Developers & Contributors

EinD Wiki

EinD's wiki contains a number of links, hints and guides for EinD users and developers.

Developer Documentation

Visit the developer documentation page for an up-to-date Doxygen documentation of EinD's code.

Clang Report

Clang's static analyzer is executed periodically on the EinD codebase. A list of current issues is published in the static analysis report.

Coverity Report

A list of issues found by Coverity is available at the EinD Coverity project. To get access, have a look at the hints on the EinD wiki.

Translation Report

An overview of the current status of translations in EinD is published on the translation report page.
